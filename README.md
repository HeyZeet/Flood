This is my first ever game I've developed. It's been a fun/frustrating process and I'm excited to see where the project goes.

Before I had made this version of the project, I had an older version that had a lot of systems clumped together in area.

This Newer project is wayy cleaner albeit with fewer systems in place currently.

The following systems are currently being worked on/stable state:

Building(WIP)(Stable)
  - Can Place blocks in the scene
  - Blocks Snap to a grid
  - Blocks can not collide with one another when being placed
  - Shows a Preview of the GameObject being placed
  - Shows Preview as "Green" if valid placement
  - Shows Preview as "Red" if invalid placement
  - Allows Cycling between pieces via mouseinput
  - "BuildPieces" have there own data such as "health" "weight" "cost"

Player(WIP)(Stable)
  - Has an inventory system
